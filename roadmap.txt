# Domaj - Roadmap

## Version 1.0 (MVP) - En cours
- [x] Interface web avec liste des conteneurs par serveur
- [x] Comparaison d'images (hash digest same tag)
- [x] Comparaison avec latest tag
- [x] Notifications par email (rapport périodique)
- [x] Agent distant léger
- [x] Déploiement Docker Compose

## Priorité — UX Dashboard
- [x] Distinction claire entre patch (same tag) et nouvelle version (latest) dans le tableau de mises à jour
- [x] Ne pas afficher les conteneurs à jour dans le dashboard (uniquement ceux nécessitant une action)

## Version 1.1 - Notifications Avancées
- [x] Intégration Telegram (API Bot via reqwest)

## Version 1.2 - Registres Privés & Database
- [ ] Support PostgreSQL comme base de données externe
- [ ] Migration SQLite vers PostgreSQL
- [ ] Migration sqlx → SeaORM (ORM async)
- [x] Support Harbor
- [x] Gestion des credentials par registre

## Version 1.3 - Actions Automatiques
- [x] Déclenchement de mise à jour depuis l'interface
- [x] Pull automatique d'images
- [ ] Restart de conteneurs
- [ ] Rollback vers version précédente
- [ ] Dry-run mode (simulation)

## Version 1.4 - Authentification Avancée
- [ ] Support OIDC/OAuth2
- [ ] Intégration Keycloak
- [ ] Intégration Auth0
- [ ] Gestion des rôles (admin/viewer)
- [ ] Audit logs des actions

## Version 2.0 - Sécurité & Observabilité
- [ ] TLS mutuel (mTLS) serveur↔agent avec PKI (serveur de certificats)
- [ ] Centralisation des credentials registre sur le serveur (envoi à l'agent lors des updates)
- [ ] Scan CVE avec Trivy
- [ ] Dashboard de vulnérabilités
- [ ] Agrégation des changelogs entre versions
- [ ] Métriques Prometheus
- [ ] Health checks avancés
- [ ] Alertes sur conteneurs unhealthy

## Version 2.1 - Organisation
- [ ] Groupes logiques (prod/staging/dev)
- [ ] Tags personnalisés sur conteneurs
- [ ] Filtres et recherche avancée
- [ ] Export des rapports (PDF/CSV)

## Hors MVP
